Book Library:

Library Management System (Frontend)

A simple Library Management System built using HTML, CSS, and JavaScript.
This project supports role-based login, book management, search functionality, and localStorage-based persistence — all without a backend.

. Features
. Authentication

.Secure login system using predefined users

.Auto-login using localStorage

.Logout functionality

. Role-Based Access
.Admin

.View all registered users

.See user details (name, email, role, phone)

.View total number of books issued per user

.Search users by name, username, or role

.Students / Teachers

.View personal library

.Add new books

.Edit existing books

.Delete books

.Search books by title or author

.View due dates and book categories

Book Management:

-Preloaded books for each user

-Custom book cover images with fallback image

-Book details include:

-Title

-Author

-Category

-Due Date

-Image

 Data Storage:

-Uses Browser Local Storage

-Books are saved per user

-Data persists after page reload

 Tech Stack:

-HTML5 – Structure

-CSS3 – Styling & Layout

-JavaScript (Vanilla JS) – Logic & Interactivity

-LocalStorage API – Data Persistence

 Highlights:

-No backend required

-Beginner-friendly JavaScript project

-Clean UI with background images

-Perfect for mini-projects, college assignments, and demos
